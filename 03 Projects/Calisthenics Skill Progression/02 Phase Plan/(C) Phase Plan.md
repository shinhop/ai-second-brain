# Phase Plan — Calisthenics Skill Progression

> Living document. Update this as phases transition or the plan flexes.
> Source reasoning lives in the top-level `GOALS.md` — this is the working copy for day-to-day tracking.

## Current Level (baseline: 2026-07-04)
* **Handstand:** Comfortable with wall drills (chest-to-wall and back-to-wall). Freestanding attempts without fear, but inconsistent. Longest freestanding hold: 9s on parallettes.
* **Muscle Up:** One kipping muscle up achieved. Several band-assisted reps possible. Not acquiring the skill — fixing timing/technique to string together more than one rep.
* **L-Sit:** Tuck L-sit held 15-20s. Core endurance is decent; real limiter is hamstring/hip flexibility (currently poor). Target: full L-sit on parallettes.

**Constraints:** poor hamstring/hip flexibility (limits L-Sit), wrists occasionally sore (manageable with parallettes), straight-arm/overhead pressing strength a confirmed weak point — flagged by coach, shows up specifically as difficulty pressing to handstand (bent-arm pressing, e.g. dips, is not the gap — that's solid). Addressed via gym's Handstand/Planche programming, not L-Sit supplemental work; becomes the central focus at Phase 2.

**Training capacity:** 4x/week gym (1hr, programmed around Back Lever/Front Lever/Handstand/Planche, L-Sit strength slotted in) + 5x/week 10-15 min at-home mobility work (2 fixed rest days, not ad hoc).

## Why L-Sit First
- The stated goal is "Handstand *presses*," not just the static hold — presses are a compression/straight-arm strength movement, which is exactly what L-Sit → straddle L-sit → full L-sit builds. This is the direct path to the press, not a detour.
- The named limiter (hip/hamstring flexibility) is a mobility gap that hasn't been trained yet — needs deliberate work, not just more reps.
- Muscle Up is a technique fix, not a new-skill acquisition — low volume, drill-focused, doesn't need to compete for primary focus.

## Phase 1 — ACTIVE (target: ~8-10 weeks from 2026-07-04)
**Primary: L-Sit**
- Strength: 4x/week, tuck → single-leg → straddle progression on parallettes + compression/hollow body work, slotted into gym sessions
- Mobility: 5x/week, 10-15 min hamstring/hip flexor routine (2 fixed rest days) — this is the actual bottleneck, don't let it slide below 5x

**Maintenance: Muscle Up**
- 1-2x/week, low volume, drill-focused (false grip holds, transition-specific drills)
- Goal is fixing timing, not building new strength

**Passive: Handstand**
- Keep leveraging existing gym programming
- Add short, frequent freestanding attempts on non-gym days (~5 min) — frequency over grinding for a new max hold

## Phase 2 — NOT STARTED
**Trigger:** tuck → straddle L-sit achieved, OR ~8-10 weeks elapsed (whichever comes first, but see checkpoint below)
- Shift primary focus to freestanding Handstand consistency
- Begin tuck handstand press negatives, leveraging Phase 1 compression strength
- Muscle Up maintenance continues unchanged
