# Hamstring & Hip Mobility Routine

> 5x/week, 10-15 min, 2 fixed rest days (pick them deliberately, don't let them float). Ties to the named bottleneck in `02 Phase Plan/(C) Phase Plan.md`: poor hamstring/hip flexibility, not core endurance.
> Strength/skill work is trained separately — see `(C) L-Sit Strength Progression.md`. This file is mobility only.

## Why this isn't solved by more L-sit reps
This is the actual limiter. It needs dedicated, daily, deliberate work — including *active* flexibility (strength at end-range), not just passive stretching. Passive-only flexibility gives you the range but not the ability to use it while compressed in an L-sit.

## 1. Warm-up (~2 min)
Don't stretch cold.

| Movement | Sets | Reps/Time | Rest |
|----------|------|-----------|------|
| Leg swings (front-back) | 1 | 10 reps each leg | — |
| Leg swings (side-side) | 1 | 10 reps each leg | — |
| Hip circles | 1 | 10 each direction | — |
| Ankle rolls | 1 | 10 each foot | — |

## 2. Passive Hamstring/Hip Work (~5-6 min)

| Stretch                                                       | Sets       | Working time | Rest between sets |
| ------------------------------------------------------------- | ---------- | ------------ | ----------------- |
| Standing or seated pike stretch                               | 2          | 45-60s       | 20-30s            |
| PNF contract-relax pike (contract 5s → relax/sink deeper 20s) | 3 per side | 25s per rep  | 15s               |
| Couch stretch (hip flexor)                                    | 2 per side | 45-60s       | 20-30s            |

- Pike stretch: breathe into it, don't bounce.
- Couch stretch: hip flexor tightness restricts the top of the L-sit position as much as hamstrings restrict the bottom — don't skip it in favor of just hamstring work.

## 3. Straddle-Specific Mobility (~3-4 min)
Since straddle L-sit is the actual Phase 1→2 trigger, this gets dedicated volume rather than being folded into general pike work.

| Stretch                | Sets | Working time | Rest between sets |
| ---------------------- | ---- | ------------ | ----------------- |
| Straddle pike stretch  | 2    | 45-60s       | 20-30s            |
| Frog/butterfly stretch | 2    | 45-60s       | 20s               |

## 4. Active Flexibility (~2-3 min)
The part passive stretching alone won't give you — trains the nervous system to use range under tension, which is what an L-sit actually demands.

| Exercise                                                                                        | Sets | Working time / reps | Rest between sets |
| ----------------------------------------------------------------------------------------------- | ---- | ------------------- | ----------------- |
| Active pike compressions (seated, pull toes toward shins while actively lifting legs off floor) | 3    | 10s hold            | 30s               |
| Standing active pike raises (kick-through style, controlled, no momentum)                       | 3    | 5 reps per side     | 30-45s            |

## Progress tracking
Log stretch depth/feel and any active-flexibility reps hit in `00 Session Logs/`. This doesn't need a separate template — a short daily note is enough; the weekly review is where it gets checked against progress on the straddle L-sit trigger.

## Notes
- If a gym session is running long or energy is low, this block should never get cut on a scheduled day — it's cheap, and it's the actual limiter. The strength block can flex; this one shouldn't drop below the committed 5x/week.
- If frequency needs to come down further at some point, that's a Phase Plan-level decision (flagged at the Week 4 checkpoint), not something to quietly erode day-to-day.

## Home Equipment Note — Yoga Blocks
- **Couch stretch:** place the back knee on a block if the floor is uncomfortable, or use one under the front foot to fine-tune hip angle.
- **Standing/seated pike stretch:** sitting on a block (raising the hips slightly) can help if hamstring tightness is currently pulling you into a rounded lower back — makes the stretch target the hamstring instead of the spine.
- **Not a dip substitute:** don't use them as a raised support for dips — foam/cork blocks aren't rigid or grippy enough to trust under full bodyweight in a dip's loaded, slightly unstable bottom position. That's what the pull-ins swap above is for instead.
