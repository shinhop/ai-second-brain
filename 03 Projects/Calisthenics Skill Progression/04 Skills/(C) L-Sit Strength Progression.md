# L-Sit Strength Progression

> Built for Phase 1 (active). Ties directly to the baseline in `02 Phase Plan/(C) Phase Plan.md` — read that first if you haven't.
> Baseline (2026-07-04): tuck L-sit 15-20s. Wrists occasionally sore (parallettes help). Overhead/straight-arm pressing strength is a confirmed weak point (coach-flagged, shows up pressing to handstand) — bent-arm pressing is not affected, and this gap is the gym's Handstand/Planche programming's job, not this file's.
> Mobility work is trained separately — see `(C) Hamstring-Hip Mobility Routine.md`. This file is strength/skill only.

## Where this fits
Gym days, 4x/week (committed number for calendaring — tied to your existing gym sessions, not extra time). Slot into your existing programmed session (10-15 min), on parallettes.

## Progression Ladder
Move to the next step only when the trigger is hit — don't skip ahead on enthusiasm.

1. **Tuck L-sit** (current step)
2. **Single-leg extended L-sit** (alternate legs across sets) — bridges tuck → straddle by loading one hip open at a time
3. **Straddle L-sit** — **this is the Phase 1→2 trigger condition** per the Phase Plan
4. **Full L-sit** (legs together, extended)

## Main Work — current step: Tuck L-sit

| Set | Working time (target) | Rest after |
|-----|------------------------|------------|
| 1   | Max hold               | 90s        |
| 2   | Max hold               | 90s        |
| 3   | Max hold               | 90s        |
| 4   | Max hold               | —          |

- **Sets:** 4
- **Rest between sets:** 90s
- **Progression trigger:** once you hit 30s+ on a set, consistently, across 2 separate sessions, move to the next ladder step
- Currently at 15-20s — don't rush this. Chasing the next step before the current hold is solid just relocates the weak point.

When you advance a ladder step, keep the same 4 sets / 90s rest structure and reset the working-time target to whatever's achievable at that new step (it'll likely drop — that's expected, log it, don't panic).

## Supplemental Compression/Support Work (2-3x/week, same session)

| Exercise                                           | Where       | Sets | Working time / reps                  | Rest between sets | Purpose                                                                                                                                                                                                                                 |
| -------------------------------------------------- | ----------- | ---- | ------------------------------------ | ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Hanging or lying leg raises                        | Either      | 3    | 8-12 reps                            | 75s               | Hip flexor strength to hold legs up, separate from hamstring length                                                                                                                                                                     |
| Hollow body hold                                   | Either      | 3    | 20-30s                               | 60s               | Trunk stability that keeps the L-sit from collapsing                                                                                                                                                                                    |
| Parallette dips — weighted, or switch to ring dips | Gym only    | 3    | 8-12 reps (add load once 12 is easy) | 90s               | Bent-arm support/compression strength for the L-sit hold. **Not** a fix for overhead pressing-to-handstand strength — that's a confirmed separate gap (coach-flagged), owned by the gym's Handstand/Planche programming, not this block |
| Parallette L-sit pull-ins (tuck-ups) — from tuck or straddle support, pull knees to chest and extend back out | Home (no dip depth) | 3    | 8-12 reps | 90s | Compression strength — the concentric fold at the hip, directly, using only the parallettes you already have. Arguably more L-sit-specific than dips since it scales with your ladder step (harder from straddle than tuck) |

Do **one or the other per session**, matched to where you're training — not both on the same day. Same sets/rest either way, so nothing else about the block changes.

## Progress tracking
Log actual numbers (ladder step, working time hit, rest actually taken) in `00 Session Logs/` using `(C) L-Sit Session Log Template.md`. Weekly review checks this against the Phase Plan targets — don't rely on memory/feel for whether the 30s trigger has actually been hit.

## Notes
- Wrists sore occasionally: parallettes (neutral grip) already mitigate this — if soreness persists, drop volume here before pushing through pain.
- If a gym session is running long or energy is low, this block (strength) is the one that can flex with gym programming — the mobility block should not (see the mobility file).
