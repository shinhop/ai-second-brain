# Commands & Skills

Quick reference for all available skills and commands in this project.

## Skills (in 04 Skills/)
- **(C) L-Sit Strength Progression.md** — Gym-day strength progression ladder (tuck → single-leg → straddle → full L-sit), with sets/working-time/rest specified for main work and supplemental compression exercises.
- **(C) Hamstring-Hip Mobility Routine.md** — Daily 10-15 min mobility block targeting the hamstring/hip flexibility bottleneck, with sets/working-time/rest specified per stretch.
- **(C) L-Sit Session Log Template.md** — Fill-in template for L-Sit strength sessions (matches `(C) L-Sit Strength Progression.md`). Sets/rest pre-filled since those are fixed by the routine; only actual working time/hold and notes are left blank. Copy into `00 Session Logs/` per session and fill in — light enough for phone use.
- **(C) Mobility Session Log Template.md** — Same format for `(C) Hamstring-Hip Mobility Routine.md` — sets/rest pre-filled per stretch, actual working time and notes left blank.
